期待値の定義

期待値とは、確率変数がとる値の平均的な値のことです。確率論や統計学では、期待値は特定の状況で「平均」的な結果を表す指標として用いられます。

期待値は通常、確率変数をXとしたときにE(X)で表され、確率分布に基づいて計算されます。

例題

例題1：サイコロの期待値

サイコロを1回振ったときに出る目の期待値を求めます。

解: サイコロの出る目は1から6までの整数です。各目が出る確率は同じなので、期待値E(X)は次のように計算されます：

E(X) = (1 + 2 + 3 + 4 + 5 + 6) / 6 = 3.5

例題2：偏ったコインの期待値

偏ったコインで、表が出る確率が0.7、裏が出る確率が0.3のとき、表が出る回数の期待値を求めます。

解: 表が出る回数をXとすると、期待値E(X)は以下のように計算されます：

E(X) = 1 * 0.7 + 0 * 0.3 = 0.7

例題3：2つのサイコロの和の期待値

2つのサイコロを振ったときに出る目の和の期待値を求めます。

解: 各サイコロの期待値は3.5なので、和の期待値は 3.5 + 3.5 = 7 です。
